# bidiwave
wavetable synth for monome norns

# requirments
norns, MIDI

# documentation
enc 1 -> change pages
key 2 and key 3 -> navigate sections
enc 2 -> select value
enc 3 change value

wavetable must be 512 samples
arpeggiator can be freezed to continue play in background, just select "freez" while playing
